    PixelArt Zeichnen


    Ich habe PixelArt mit HTML/CSS + Javascript programmiert. Sie haben ein leeres Raster (8*8). Sie können eine Farbe auswählen und beginnen, die Pixel zu füllen. So können sie ein Bild zeichnen. Mit weiss können sie die Pixelfarben löschen. Seien Sie kreativ !!!

    Colorgame

    Colorgame spielen


    Ich habe ein Farbenspiel mit HTML/CSS und Javascript programmiert. Die Farbe wird in RGB-Notation angezeigt und sechs Farben zur Auswahl zugeordnet. Sie müssen die richtige Farbe wählen. Wenn Sie die richtige Farbe gefunden haben, aktualisieren Sie die Website, um neu zu starten. Viel Spaß! :)
